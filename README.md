# Rehbar

**Rehbar** (راہبر — *guide*) is a web app that helps people discover and access community resources across health, education, legal aid, finance, and more.

## Features

- Browse resources by category (Health, Education, Legal, Finance, Community)
- Search resources by title, description, or category
- Clean, responsive UI that works on mobile and desktop

## Getting Started

### Prerequisites

- A modern web browser
- [Node.js](https://nodejs.org/) (optional, for the local dev server)

### Running Locally

Open `index.html` directly in your browser, **or** serve it with the built-in dev server:

```bash
npm start
```

The app will be available at `http://localhost:3000` (or the port shown in your terminal).
